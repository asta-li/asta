# Hi

This is my personal website.

## Development

1. **Clone the repo.**

```sh
git clone https://github.com/asta-li/asta-li.git
```

2. **Develop locally.**

Start the website locally.
```sh
cd asta-li
gatsby develop
```
View the website at `http://localhost:8000`.

3. **Deploy.**

Create production build.
```sh
gatsby build
gatsby serve
```

Deploy to Google App Engine.
```sh
gcloud app deploy
```
